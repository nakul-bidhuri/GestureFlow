\# GestureFlow 🚀



GestureFlow is a browser-based hand gesture recognition web application built using HTML, CSS, JavaScript, TensorFlow.js, and MediaPipe.



\## Features



\* ✋ Real-time hand gesture detection

\* 🎯 Multiple gesture recognition

\* 📷 Webcam integration

\* ⚡ Fast browser-based processing

\* 🔒 Privacy-friendly (runs locally in browser)

\* 📊 Live gesture status and analytics

\* 🎨 Modern responsive UI



\## Technologies Used



\* HTML5

\* CSS3

\* JavaScript

\* TensorFlow.js

\* MediaPipe Hands



\## Supported Gestures



\* Open Palm

\* Fist

\* Victory Sign

\* Thumbs Up

\* Pointing

\* OK Sign



\## Installation



1\. Clone the repository



```bash

git clone https://github.com/nakul-bidhuri/GestureFlow.git

```



2\. Open the project folder



```bash

cd GestureFlow

```



3\. Open `index.html` in your browser
